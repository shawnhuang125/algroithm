### 何謂圖?
- 圖 = 表示法`G(V,E)`,翻譯是圖是一個包括點集合和邊集合的集合
- 依據圖的特性可以衍伸出有向圖,無向圖,權重圖
- 有向圖:單條線有單向箭頭的圖皆代表有向圖。
- 無向圖:單條線有兩個方向的箭頭或是沒有箭頭的圖皆代表為無向圖。
- 權重圖:邊上有數字
